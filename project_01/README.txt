This program:
	Plays a frequency
	Uses a potentiometer to choose frequency of standard guitar tuning EADGBE
	Uses buttons to turn a servo to adjust guitar tuning

To use:
	1. Wire on breadboard according to fritzing diagram
		*Power provided via micro USB to pocketbeagle 
	2. Make sure ht16k33.py is in the same directory as the tuner.py file
	4. If you desire to use an alternate tuning, edit the two dictionaries to the notes/frequecies
	of your choice 
	3. Run the program
